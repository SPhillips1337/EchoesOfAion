Echoes of Aion - Turn Submission

Submit Turn Actions

Enter a JSON array of TurnAction objects. Each action needs: type, payload (object), and turnNumber (positive integer).

Response

Valid Action Types:

  • MOVE_FLEET: payload needs fleetId, destinationStarId
  • COLONIZE_PLANET: payload needs fleetId, planetId
  • BUILD_STRUCTURE: payload needs planetId, structureType
  • CONSTRUCT_SHIP: payload needs planetId, shipType
